# Trust Assets

This directory contains verification proof images for the Technology Provider status.

## Required Files

Add sanitized screenshots of your Technology Provider verification:

```
technology-provider-proof-1.png   (Dashboard status / onboarding completion)
technology-provider-proof-2.png   (API access confirmation)
technology-provider-proof-3.png   (WhatsApp Business Platform integration)
```

These images are displayed in:
- `/trust` page — Full carousel with all 3 images, prev/next navigation, and thumbnails
- Homepage modal — First image only, with "View all evidence (3)" link

## Guidelines

- Use a 16:9 aspect ratio for best display
- Sanitize any sensitive information (access tokens, account IDs, etc.)
- Keep each file size under 500KB for optimal loading
- Images gracefully fallback to placeholders if missing
